# i18n

## Built-in Locales

| Code | Language |
|------|----------|
| `en` | English |
| `zh-TW` | Traditional Chinese |

```bash
npx mdsone --locale en
```

## Locale File Structure

```text
locales/
├── en.json
└── zh-TW.json
```

```json
{
  "cli": {
    "err_md_dir_missing": "Source directory not found: {dir}"
  },
  "template": {
    "html_lang": "en",
    "footer_label": "Build date: {BUILD_DATE}"
  }
}
```

| Section | Purpose |
|---------|---------|
| `cli` | Tool logs and error messages |
| `template` | UI text embedded in HTML |

## Adding a New Locale

```bash
# Windows PowerShell
Copy-Item locales/en.json locales/ja.json

# macOS / Linux
cp locales/en.json locales/ja.json

# Edit, then run
npx mdsone --locale ja
```

If the specified locale is not found, it automatically falls back to `en`.

## Multi-language Mode (I18N_MODE)

### Enabling

In `config.toml` (recommended):

```toml
[i18n]
mode = true
default_locale = "en"
```

Or via environment variables:

```env
I18N_MODE=true
DEFAULT_LOCALE=en
```

### Source Directory Structure

Under the source folder, use `[locale]`-named subdirectories to separate documents by language:

```text
docs/
├── [en]/
│   ├── 01_intro.md
│   └── 02_usage.md
└── [zh-TW]/
    ├── 01_intro.md
    └── 02_usage.md
```

> Directory names must use the bracket `[locale]` format, e.g. `[en]`, `[zh-TW]`, `[ja]`.

### Running

```bash
npx mdsone --source ./docs
```

The tool will automatically scan all `[locale]` subdirectories and bundle all language documents into a single HTML file.

### Output Behaviour

- The generated HTML includes a built-in language switcher dropdown (`<select>`) in the top-right corner
- Switching languages updates the page instantly without a full reload
- The user's language preference is saved in `localStorage` and automatically applied on the next visit
- `default_locale` (config.toml) or `DEFAULT_LOCALE` (env) sets the language displayed on first load

### Single-language vs Multi-language Data Structure

| Mode | `window.mdsone_DATA` Structure |
|------|-------------------------------|
| Single-language (default) | `{ docs: [...], config: {...}, i18n: {...} }` |
| Multi-language | `{ locales: [...], defaultLocale: "...", docs: { "en": [...], "zh-TW": [...] }, config: {...}, i18n: { "en": {...}, "zh-TW": {...} } }` |
