# Plugins

Built-in plugins:

- `image`
- `katex`
- `shiki`
- `copy`
- `line_number`
- `minify`

Pipeline summary:

1. Core renders Markdown to HTML.
2. `PluginManager.processHtml()` runs DOM plugins in order.
3. `PluginManager.getAssets()` collects plugin CSS/JS.
4. `PluginManager.processOutputHtml()` runs final output hooks (minify stage).

## image

Purpose:

- Convert `<img src="...">` to base64 data URL.
- Supports local files and remote `http(s)` URLs.
- Optional resize/compress (requires `sharp`).

CLI:

```bash
npx mdsone README.md --img-embed=base64
npx mdsone README.md --img-embed=base64 --img-max-width 400 --img-compress 80
```

TOML:

- `[plugins.image].embed` (`off` or `base64`)
- `[plugins.image].max_width`
- `[plugins.image].compress`

## katex

Purpose:

- Registers `markdown-it-katex` in markdown stage.
- Inlines KaTeX CSS into final HTML.

CLI:

```bash
npx mdsone README.md --katex
npx mdsone README.md --katex=full
```

TOML:

```toml
[plugins.katex]
enable = true
mode = "woff2"   # or "full"
```

## shiki

Purpose:

- Rewrites `<pre><code>` blocks using Shiki highlighting.
- Uses `highlight.js` only for auto-detect fallback when language is missing.

CLI:

```bash
npx mdsone README.md --code-highlight=off
```

TOML:

- `[plugins.shiki].enable`

Template-controlled options (`template.config.json`):

- `config.types.<name>.code.Shiki.dark`
- `config.types.<name>.code.Shiki.light`
- `config.types.<name>.code.Shiki.auto_detect`

## copy

Purpose:

- Adds code copy button behavior.
- Supports line-based and command-section copy modes.

CLI:

```bash
npx mdsone README.md --code-copy=off
npx mdsone README.md --code-copy=line
npx mdsone README.md --code-copy=cmd
```

TOML:

- `[plugins.copy].enable`
- `[plugins.copy].mode` (`off`, `line`, `cmd`)

## line_number

Purpose:

- Adds line numbers to code blocks.
- Compatible with wrapped structures generated by other plugins.

CLI:

```bash
npx mdsone README.md --code-line-number
npx mdsone README.md --code-line-number=off
```

TOML:

- `[plugins.line_number].enable`

## minify

Purpose:

- Minifies final full-page HTML (including inline CSS/JS) via `html-minifier-terser`.

CLI:

```bash
npx mdsone README.md --minify
npx mdsone README.md --minify=off
```

TOML:

- `[plugins.minify].enable`

## Plugin Order

Default order:

1. `image`
2. `katex`
3. `shiki`
4. `copy`
5. `line_number`
6. `minify` (output stage; always last)

You can override order with `[plugins].order`.
