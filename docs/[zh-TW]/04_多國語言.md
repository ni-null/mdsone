# 多國語言（i18n）

多國語言功能只有在 `--i18n-mode` 模式下才會啟用。
單語模式可用 `--locale` 指定 UI 語系，但不會載入多語內容。

## 檔案結構

### 1. UI 語系檔（locales）

```text
locales/
├── en.json
└── zh-TW.json
```

```json
{
  "cli": {
    "err_md_dir_missing": "找不到來源資料夾：{dir}"
  },
  "template": {
    "html_lang": "zh-TW",
    "footer_label": "建置日期：{BUILD_DATE}"
  }
}
```

### 2. 多語內容資料夾（[locale]）

```text
docs/
├── [zh-TW]/
│   ├── 01_intro.md
│   └── 02_usage.md
└── [en]/
    ├── 01_intro.md
    └── 02_usage.md
```

> 子目錄名稱必須使用方括號 `[locale]` 格式，例如 `[zh-TW]`、`[en]`。

## 多國語言相關參數

### CLI

| 參數 | 說明 | 範例 |
|------|------|------|
| `--i18n-mode` | 啟用多國語言模式（布林旗標） | `--i18n-mode` |
| `--i18n-default CODE` | 預設顯示語系（僅在 i18n 模式生效） | `--i18n-default zh-TW` |
| `--locale CODE` | UI 語系（單語模式） | `--locale zh-TW` |

```bash
# 多國語言模式（自動走合併邏輯）
mdsone ./docs --i18n-mode --i18n-default zh-TW -o dist/index.html
```

> 注意：啟用 `--i18n-mode` 會自動採用合併模式（等同 `-m`），
> 輸出為單一 HTML 檔案。

### 環境變數

```env
I18N_MODE=true
DEFAULT_LOCALE=zh-TW
```

### config.toml

```toml
[i18n]
mode = true
default_locale = "zh-TW"
```

## 預設參數

- `i18n.mode` 預設為 `false`。
- `i18n.default_locale` 預設為空值。
- i18n 模式下若未指定 `default_locale`，會依序嘗試：
  `i18n.locale` → 第一個可用語系 → `en`。
- `i18n.locale`（單語模式 UI 語系）預設為 `en`。

## 多語輸出行為

- 產生的 HTML 右上角內建語言切換選單。
- 切換語系時頁面即時更新，不需重新載入。
- 使用者選擇的語系會儲存在 `localStorage`，下次開啟自動套用。
