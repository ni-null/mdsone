# 快速開始

## 安裝

```bash
# 透過 npx 執行（推薦）
npx mdsone [選項]

# 或全域安裝
npm install -g mdsone
```

## 用法示例

### 單一 Markdown 檔案

```bash
# 輸出到目前目錄（如 README.html）
npx mdsone README.md

# 指定輸出檔案
npx mdsone README.md -o index.html
```

### 多個 Markdown 檔案（批次模式）

```bash
# 每個 .md 產生各自對應的 .html 到目前目錄
npx mdsone a.md b.md c.md

# 輸出到指定目錄
npx mdsone a.md b.md -o ./dist
```

### 資料夾（批次模式）

```bash
# 資料夾中每個 .md 各自產生對應 .html（-o 需為目錄）
npx mdsone ./docs -o ./dist
```

### 合併模式

```bash
# 合併多個檔案 → 目前目錄產生 merge.html
npx mdsone intro.md guide.md -m

# 合併多個檔案並指定輸出檔名
npx mdsone intro.md guide.md -m -o manual.html

# 合併整個資料夾
npx mdsone ./docs -m

# 合併資料夾並指定輸出路徑
npx mdsone ./docs -m -o manual.html
```

### 圖片嵌入

```bash
# 嵌入圖片為 base64
npx mdsone README.md -o index.html --img-to-base64

# 嵌入圖片並指定最大寬度
npx mdsone README.md -o index.html --img-to-base64 --img-max-width 400

# 嵌入圖片並壓縮（需要 sharp）
npx mdsone README.md -o index.html --img-to-base64 --img-max-width 400 --img-compress 80
```

### 模板與語系

```bash
# 指定模板與語系
npx mdsone ./docs -m -o output.html --template minimal --locale zh-TW
```

### 覆蓋保護

```bash
# 若輸出檔已存在則中止
npx mdsone README.md -o output.html -f false
```

## 兩種運作模式

| 模式                 | 說明                                  |
| -------------------- | ------------------------------------- |
| **批次模式**（預設） | 每個 Markdown 檔案各自產生獨立的 HTML |
| **合併模式**（`-m`） | 所有輸入合併為單一 HTML，以分頁呈現   |
