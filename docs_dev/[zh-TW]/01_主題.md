# 主題開發

## 使用前提（重要）

`--template-dev` 是「原始碼開發模式」，只能在 `git clone` 的專案目錄中使用。  
請先用 Git 取得原始碼並在專案根目錄執行：

```bash
git clone https://github.com/ni-null/mdsone.git
cd mdsone
npm install
```

以下發行型態不支援 `--template-dev`：

1. `npm i -g mdsone` 安裝版
2. Bun 編譯的單檔執行檔（release binary）

若在非原始碼環境執行，會看到：

```text
template-dev is available only in source checkout.
```

## 模板目錄結構

```text
templates/
└── my-template/
    ├── template.html
    ├── style.css
    ├── template.config.json
    ├── locales/
    │   ├── en.json
    │   └── zh-TW.json
    └── assets/
        ├── 01_base.css
        └── 02_behavior.js
```

## 快速開發流程

1. 複製內建模板作為新主題

```bash
# Windows PowerShell
Copy-Item -Recurse templates/normal templates/my-template

# macOS / Linux
cp -r templates/normal templates/my-template
```

2. 啟動模板開發模式（未指定輸入時預設 `README.md`）

```bash
npx tsx src/cli/main.ts --template-dev --template templates/my-template
```

3. 開啟終端機輸出的網址（例如 `http://127.0.0.1:xxxxx/template.html`）
4. 用 VS Code / AI 工具調整下列檔案
   - `template.html`
   - `style.css`
   - `template.config.json`
   - `assets/**`
   - `locales/**`
5. 儲存後頁面會自動重新整理，持續迭代版型

## 開發完成後驗證

用一般 CLI 重新產出一次，確認正式輸出結果：

```bash
npx tsx src/cli/main.ts ./docs -m -o ./_site/preview.html --template templates/my-template -f
```

## template.html 佔位符

| 佔位符 | 說明 |
| --- | --- |
| `{TITLE}` | 文件站點標題 |
| `{LANG}` | HTML `lang` 屬性 |
| `{CSS_CONTENT}` | `style.css` 內容 |
| `{LIB_CSS}` | 外掛注入 CSS |
| `{EXTRA_CSS}` | `assets/*.css` 內容 |
| `{LIB_JS}` | 外掛注入 JS |
| `{EXTRA_JS}` | `assets/*.js` 內容 |
| `{MDSONE_DATA_SCRIPT}` | 內嵌資料腳本 |
